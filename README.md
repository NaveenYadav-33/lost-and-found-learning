# Lost and Found Community Platform

A full-stack web application for reporting and retrieving lost or found items within a campus or community environment.

---

## 🚀 Overview

This is a MERN stack-based web application built to explore and understand full-stack development concepts.

Users can:
- Report lost items
- Post found items
- Browse listings
- Interact via comments

---

## ✨ Features

- User authentication (login, signup, password management)
- Post and manage lost/found items
- Real-time CRUD operations for posts and comments
- Image upload support using Firebase Storage
- Responsive and dynamic user interface

---

## 🛠 Tech Stack

- Frontend: ReactJS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: Firebase Authentication
- Storage: Firebase Storage
